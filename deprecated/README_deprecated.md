# Archived modules

The scripts in this folder were part of earlier exploration and are **not** part of the analysis reported in the manuscript. They are retained for completeness and transparency. No active module in `src/` imports them, and they are not required to reproduce any result or figure.

| Module | Note |
|---|---|
| `geodesic_benchmark.py` | Geodesic-distance exploration between immune and stromal compartments. Not used in the final analysis. |
| `weighted_geodesic.py` | Weighted variant of the above. Not used in the final analysis. |
| `spatial_analysis_v2_additions.py` | Extension to `spatial_analysis_v2.py` providing geodesic support; superseded. The reported spatial-gradient and map outputs are produced by `spatial_analysis_v2.py` in `src/04_caf_barrier/`. |

These files can be ignored for reproduction. They are kept so that the development history of the project is fully visible.
